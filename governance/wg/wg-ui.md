# Working Group Charter — wg-ui

Chartered by the Steering Committee under `06_GOVERNANCE.md` §3 (ratified
D-0017, `governance/decisions.md`).

- **Domain:** The ss_ui universal layout engine, themes, input methods, and accessibility across rect/square/round/e-ink/headless displays.
- **Chair:** dylan peterson `<dylan@elementalmfg.com>` — founding maintainer,
  bootstrap staffing per D-0017; replaced by SC appointment when the WG is
  staffed (S-01-009 hand-off).
- **Maintainers:** founding maintainer (bootstrap). Steady state requires the
  chair plus 2+ maintainers per `06_GOVERNANCE.md` §3; CODEOWNERS entries
  move from the maintainer to this WG's team at that point.
- **Channel:** GitHub Discussions on the repository of record (D-0011) and
  Matrix `#ss-sp:matrix.org` (interim channels per D-0012).
- **Cadence:** async-first. A written status update is posted to the WG's
  Discussions thread at least monthly while the WG has active epics, and the
  WG is reviewed at every quarterly constitutional review (S-01-016).
- **Escalation:** decisions the WG cannot close → Steering Committee via a
  `governance/decisions.md` entry (RFC first where §4.1 of
  `06_GOVERNANCE.md` requires one). Security incidents → `SECURITY.md`
  intake / SRT. Conduct issues → `CODE_OF_CONDUCT.md` enforcement ladder.
- **Disband/split:** by SC vote per `06_GOVERNANCE.md` §3.
